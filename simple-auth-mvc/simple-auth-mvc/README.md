# Simple PHP MVC Authentication

Small lab task project using PHP, MySQL, HTML, CSS, and JavaScript.

## Features

- Database design
- Database connection
- Sign up with server-side validation
- Sign in with server-side validation
- Protected dashboard
- Logout

## Run in XAMPP

1. Copy this folder to `C:\xampp\htdocs\simple-auth-mvc`.
2. Start Apache and MySQL from XAMPP Control Panel.
3. Open `http://localhost/phpmyadmin`.
4. Import `database.sql`.
5. Visit `http://localhost/simple-auth-mvc`.

Default database settings are in `models/User.php`:

- Database: `simple_auth_mvc`
- User: `root`
- Password: empty
